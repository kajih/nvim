# Workflow / Keybinds

## Standard

å - [  (swedish vimkeys)
ä - ]  (swedish vimkeys)

jk - exit insert mode
vtrl+o - back
ctrl+j - move line down with indent
ctrl+k - move line up with indent

<F2> - Telescope
<F3> - Neogit

## Splits
<leader>sv - Split vertically
<leader>sh - Split horizontally
<C-w>o - Only window (removes other splits)

<leader>{hjkl} - Split nav
<C-w> {hjkl} - split nav
<A-{hjkl}> - split nav / <A-k> seems broken

## Harpoon
Quick navigation between project files, expands LSP
<F1> - UI
<leader>pa - add file
<leader>pq - menu toggle
<leader>p1 - file 1
<leader>p2
<leader>p3
<leader>p4

## Comment
Quickly toggle comments on a section.

gc - comment block
can use gc + movement
